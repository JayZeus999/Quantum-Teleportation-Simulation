# Quantum-Teleportation-Simulation

Completed the Quantum Teleportation Simulation project.

Created a Bell pair between Alice and Bob using H and CNOT gates.

Prepared a custom quantum state on the message qubit.

Performed a Bell measurement to destroy the original state and transfer its quantum information through entanglement + classical communication.

Used conditional Pauli-X and Pauli-Z corrections to reconstruct the exact original state on Bob’s qubit.

The simulation demonstrates how quantum information can be transmitted without physically moving the qubit itself.
